首先安装[nvm](https://github.com/creationix/nvm)，这是个多版本管理器，也是目前装nodejs最优办法

```
yum install git
```

```
curl https://raw.github.com/creationix/nvm/master/install.sh | sh
```

```
echo ". ~/.nvm/nvm.sh" >> ~/.bashrc
```

关闭terminal，再打开

```
nvm install 0.10.4
```

<!--more--> 再装[npm](https://npmjs.org/)，这是用来管理模块的

```
curl https://npmjs.org/install.sh | sh
```

安装express模块

```
npm install express
```

升级模块

```
npm update
```
