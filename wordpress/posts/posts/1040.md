old9说:

XHTML 走的是 xml 分支，属于从 HTML 向 xml 过渡型的，不过由于在 web 上应用 xml 的种种弊端和不足，估计此路不通了…… 实际上现有的 XHTML1 并没有体现出其 xml 的优势，而且通常它也并没有被浏览器当作 xml 来处理，而是当作普通 HTML 来处理，从而让这种转变的努力形同虚设。

事实证明，试图在 web 上使用 xml 实在是太困难了…… xhtml2 估计会胎死腹中亦或转向其他应用方向。

所以业界领袖们打算重回 HTML，不要让 web 在 xml 的道路上勉为其难而转为重新扩展和增强 HTML，也就有了非官方的 HTML5 和官方的 new HTML Working Group。

BTW. 这和 xml+xsl 不一样，xml+xsl 最终到了浏览器显示层面，还是 HTML。
