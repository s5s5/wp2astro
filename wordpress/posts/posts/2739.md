这几天折腾下Linux，反复装了几遍，为了些小问题，有强迫症的人受不了啊~~~

下载安装 [Ubuntu Server](http://www.ubuntu.org.cn/download/server) 先中文安装，后面要改中文为英文，不然就是显示成方块，为了这个问题，我装了好几遍，NND~

```
sudo vi /etc/default/locale
```

```
LANG="en_US.UTF-8"
LANGUAGE="en_US:en"
```

```
sudo reboot
```

开启ROOT密码

```
sudo passwd root
```

升级更新，这应该是大家最常用的命令吧

```
agt-get update
agt-get upgrade
```

安装zhcon支持中文

```
agt-get install zhcon
```

最后安装 [LNMP](http://www.lnmp.org/install.html) ，折腾完了，早点睡吧……
