层是一种网页布局概念，还不是单指的ＤＩＶ这个标签，当然，也可以把层技术看做如何用好ＤＩＶ标签，因为实际上所谓层布局也是通用对ＤＩＶ标签的属性调整来实现的。层是一种 HTML 页面元素，您可以将它定位在页面上的任意位置。层可以包含文本、图像或其它任何可在 HTML 文档正文中放入的内容。而且，可以通过层的前后放置，隐藏某些层而显示其它层，以及在屏幕上移动层，来做出你想要的效果。  
　　但是，Microsoft Internet Explorer 4.0 和 Netscape Navigator 4.0 之前版本的 Web 浏览器无法显示层，而这两种浏览器显示层的方式并不完全一致。  
　　所以，在你要使用层之前，最好先调查一下浏览你的网站的浏览器的类型，还有就是浏览者的分辩率，层在不同分辩率下，会有不同的效果，当然现在国内配机时一般是１７寸的显示器，大家可以把１０２４＊７６８做为默认分辩率，同时兼顾其它分辩率。
