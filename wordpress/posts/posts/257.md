<br/>  
IE的IMG标签一直显示不了PNG图像的透明背景，前几天在mximize.com看到一个解决的方法：使用AlphaImageLoader和Alpha滤镜通过CSS的方法显示透明的PNG图片<br/>  
<br/>

<style type="text/css">  
.pngholder{  
   width:100px;  
   height:100px;  
   filter:progid:DXImageTransform.Microsoft.AlphaImageLoader(src='[/assets/uploads/attachments/200510/03\_124401\_ie.png](/assets/uploads/attachments/200510/03_124401_ie.png)');  
  }  
.pngalpha{  
     filter:progid:DXImageTransform.Microsoft.Alpha(opacity=0);  
     background:url([/assets/uploads/attachments/200510/03\_124401\_ie.png](/assets/uploads/attachments/200510/03_124401_ie.png)) no-repeat;  
     width:100px;  
     height:100px;  
  }  
</style>

  <!- And this is your code to implement the image ->  
  <div>透明</div>  
  <div class="pngholder"><div class="pngalpha"></div></div>  
  <div>不透明</div>  
  <img src="[/assets/uploads/attachments/200510/03\_124401\_ie.png](/assets/uploads/attachments/200510/03_124401_ie.png)"/>
