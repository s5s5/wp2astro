在知乎上看到一个问题 [为什么大部分程序员都喜欢用黑色界面?](http://www.zhihu.com/question/24631380/answer/28467974) 中 sunus 分享的透明IDE方案非常赞，所以我也研究了半天，终于在MAC上试验成功，接下来就是学习怎么玩VIM了…… 效果如下：

![一边打码一边看片](/assets/uploads/2014/08/QQ20140830-2-1024x640.png)

简单来说就是用设置了 透明背景 的 [iTerm2](http://iterm2.com/) 中用 VIM 来编码。 怎么设置透明背景这个很好找，就不说了。

说下比较难搞的几个点：

<!--more-->

1. VIM要想用的好，还要配置好，我用了 [spf13-vim](http://vim.spf13.com/) 来进行VIM自动化配置，然后可以有代码染色，提示等功能。
2. 但 MAC 自带的 VIM 是7.3版本，spf13-vim安装后，总是会提示有错误。
3. 所以又用 [Homebrew](http://brew.sh/index_zh-cn.html) 安装 VIM 最新版本。（你要安装最好先安装 Homebrew ，然后用他先安装 Git ，然后再安装VIM最新版本）
4. 然后还要改终端命令‘vim’指向你的最新版本VIM，这里我装了 [Oh My Zsh](http://ohmyz.sh/) 修改 **.zshrc** 文件添加以下代码：

.

```
alias vim="/usr/local/bin/vim"
```

然后，你就可以一边写代码一边看文档，提升工作效率了。
