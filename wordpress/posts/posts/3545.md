## 服务器配置

服务器有了，然后就要登录服务器进行配置，这里是重点。

### 服务器密码在哪

首先要去 [站内信](https://console.qcloud.com/message/index/all/0) 看一下你拉服务器密码，**请查收……** 这行字里有密码

![站内信](/assets/uploads/2016/11/08.png)

打开，看到我红框里的重要内容了嘛，猜猜我的密码是啥 :P

![站内信内容](/assets/uploads/2016/11/09.png)

### 登录服务器

有了密码，还要用 Bitvise SSH Client 这个软件才可以访问服务器，**下载 - 安装**

下载地址：[https://www.bitvise.com/ssh-client-download](https://www.bitvise.com/ssh-client-download)

安装好，打开，按下图把 IP、端口、用户名、密码 填写好，登录

![Bitvise SSH Client](/assets/uploads/2016/11/10.png)

登录成功，终端会显示成这样

![Terminal](/assets/uploads/2016/11/11.png)

用以下命令升级服务里的软件到最新版

```
yum update -y
```

### 安装OneinStack

[OneinStack](https://oneinstack.com/) 是一个一键安装程序，可以方便的安装 Nginx、MySQL、PHP 这些程序，有这些才能让Wordpress正常运行

在终端逐步录入执行以下命令

```
yum -y install wget screen curl python   #for CentOS/Redhat
wget http://mirrors.linuxeye.com/oneinstack-full.tar.gz    #包含源码，国内外均可下载
tar xzf oneinstack-full.tar.gz
cd oneinstack    #如果需要修改目录(安装、数据存储、Nginx日志)，请修改options.conf文件
screen -S oneinstack    #如果网路出现中断，可以执行命令`screen -r oneinstack`重新连接安装窗口
./install.sh     #注：请勿sh install.sh或者bash install.sh这样执行
```

然后按下图选择好相关软件

![Install OneinStack](/assets/uploads/2016/11/12.png)

- **注意：请记录好你的数据库密码，后面会用到！！！**
- **注意：请记录好你的数据库密码，后面会用到！！！**
- **注意：请记录好你的数据库密码，后面会用到！！！**

安装成功后，用浏览器打开你的公网IP，可以看到成功页面

![OneinStack默认页面](/assets/uploads/2016/11/14.png)

## 安装Wordpress

Wordpress就是博客程序了，他是世界上最流行的博客程序，安装方便，使用简单。

### 下载Wordpress

进入你域名目录

```
cd /data/wwwroot/www.youdomain.com/
```

或者OneinStack默认目录

```
cd /data/wwwroot/default/
```

下载Wordpress

```
wget https://cn.wordpress.org/wordpress-4.7-zh_CN.tar.gz
```

解压

```
tar xzf wordpress-4.7-zh_CN.tar.gz
```

### 建立数据库

用浏览器打开你的公网IP，点击 phpMyAdmin，用之前记录的root账号密码登录数据库

![phpMyAdmin](/assets/uploads/2016/11/15.png)

点击 _New_ 新建数据库

- **数据库名 wordpress**
- **字符集选 utf8mb4_unicode_ci**

![新建数据库](/assets/uploads/2016/11/16.png)

### 安装Wordpress

用浏览器打开你的公网IP，在地址后添加 /wordprss 访问，比如

```
http://19.99.67.78/wordpress/
```

访问后会显示以下界面，点击 现在就开始！

![准备安装wordpress](/assets/uploads/2016/11/17.png)

会提示无法创建 wp-config.php 文件，这时 全选 - 复制

![无法创建 wp-config.php](/assets/uploads/2016/11/18.png)

在终端进入wordpress目录，这里以默认目录为例

```
cd /data/wwwroot/default/wordpress/
```

然后用vi创建 wp-config.php

```
vi wp-config.php
```

在 vi 中，按字母 i 进入编辑模式，复制进之前粘贴板中的内容

![vi](/assets/uploads/2016/11/19.png)

然后按 ESC，然后输入 :wq 保存退出

然后再回到浏览器中，进入下一步，设置wordpress **用户名 和 密码，请保存好啊！！！**

![安装wordpress](/assets/uploads/2016/11/20.png)

然后会安装成功，提示登录

![登录](/assets/uploads/2016/11/21.png)

登录成功啦！！！这就是wordpress的后台

![后台](/assets/uploads/2016/11/22.png)

这就是安装好的wordpress了

![前台](/assets/uploads/2016/11/23.png)
