![](https://s5s5.me/wp-content/themes/smells-like-facebook/screenshot.png)这几天突然发现之前用的主题 [zBench](http://wordpress.org/extend/themes/zbench) 用的人好多啊（毕竟还有段时期是世界下载第一……），有点不爽，为啥和别人一样呢？再一想，要么彻底得与别人不一样，要么就彻底的与别人一样！于是想到目前全球最火的SNS网站就是Facebook了，搞个这样的主题绝对是与全球最多人类是站在一起的，虽然可能大家会有点审美疲劳但还是有一定的亲和力~

接下来去WP官方主题站搜索一番，竟然只发现有一套类FB主题，而且安装之后网站就打不开了，只好上FTP把主题删除掉……以我小人之心猜测可能是WP怕FB抢了他们的用户，所以对所有类FB主题进行了限制。不过 [大猫](http://ooxx.me/) 认为这是国外版权保护比较厉害，WP怕FB和他打界面版权之类的官司，所以限制了这一类的主题。不管什么原因，总之在官网上找主题之路是行不通的，所以谷歌百度搜搜搜狗有道BING了一番之后，当当当当！！！！终于找到了现在这套《[Smells Like Facebook](http://smells-like-facebook.nazieb.com/)》。 <!--more--> 废话一下，这个主题作者已经停止更新了，作者说：

> **Smells Like Facebook Officially Suspended** Hi, It’s very sad to me to tell you that Smells Like Facebook is now officially removed from WordPress Theme Directory. The removal took effective today, therefore you can’ t download it anymore from the Directory. Actually the theme has been suspended from updating since 2009. That makes me unable to update the theme to newer version, but the download link is still available for you (for theme version 2.4). But today, the theme page itself is not available to access. You’ll be redirected to Directory main page every time you access Smells Like Facebook page.

废话结束。这套主题不只是界面上就闻起来像FB，在功能上更是像，特别是点了站内的任何链接，页面和FB一样不是整个页面跳转，而是文章显示部分进行局部刷新哦（不信，你点几个链接试试）。还有什么“URL Hash”功能（为解决网站AJAX化后产后的URL变化问题的），搞的网站URL里有一堆“#”号，搞得和最近流行给URL里加“!”和“#”一样，哈哈。

当然现在给WP上装的插件和主题自带的JS和CSS都很多，每次打开时请求特别的多，导致网站很慢，还好 [大猫](http://ooxx.me/) 又推荐用 [WP Minify](http://omninoggin.com/wordpress-plugins/wp-minify-wordpress-plugin/) 用合并了JS和CSS文件，当然这个过程比较麻烦，我调试了比较长的时间，因为并不是所有JS和CSS合并后网站就一定不出错的。最后再用 [WP Super Cache](http://ocaoimh.ie/wp-super-cache/) 做下CACHE，HOHO，大功告成。

最后，欢迎大家来点击测试本站，有BUG你说话！
